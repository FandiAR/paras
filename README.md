## How to run
1. Clone this repo with following commands:

    ```
    git clone https://github.com/FandiAR/batixoft.git
    ```
2. Move to batixoft directory 

    ```
    cd batixoft
    ```
3. Run npm install 

    ```
    npm install
    ```
4. Run apps 

    ```
    npm run storybook
    ```
5. Open browser with the following url 

    http://localhost:6006
